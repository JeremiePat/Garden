### Stocker des changements

```bash
# Stocker un état
$ git stash
```

```bash
# Appliquer un état stocké
$ git stash apply (<id>)
```

```bash
# Supprimer un état stocké
$ git stash drop (<id>)
```

```bash
# Les deux précédent en une commande
$ git stash pop (<id>)
```
