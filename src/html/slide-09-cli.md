### Créer des dossiers : mkdir

```bash
# Créer un dossier dans le dossier courant.
$ mkdir <dossier> …
```

```bash
# Créer un dossier et toute l'arboressence nécessaire si elle n'existe pas.
$ mkdir -p <chemin> …
```

```bash
# Et pour créer un fichier vide?
$ touch <fichier> …
```
