### Gérer les branches

```bash
# Créer une branche
$ git branch <nom>

# Supprimer une branche
$ git branch -d <nom>
```

```bash
# Changer de branche
$ git checkout <branche>
```

```bash
# Fusionner la branche selectionnée dans la branche courante
$ git merge <branche>
```

```bash
# Changer le point de divergence de la branche courante
# sur la tête (HEAD) de la branche choisie
$ git rebase <branche>
```
